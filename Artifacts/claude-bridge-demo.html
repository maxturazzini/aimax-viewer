Claude Bridge Demo

Test all 4 modes of the /__claude API endpoint

terminal

Terminal Mode

Opens an interactive Claude session in the VS Code terminal with your prompt pre-filled. Claude runs interactively — you can continue the conversation.

vscode

VS Code Mode

Opens the Claude Code extension panel and starts a new conversation. Use this to work with Claude in the GUI.

print

Print Mode

Sends the prompt to Claude CLI and returns the response as JSON. The answer appears inline below — no context switch needed.

copy

Copy Mode

Copies the prompt to your clipboard. Simple utility for when you want to paste it elsewhere.
