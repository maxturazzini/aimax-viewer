Quick Setup

Follow these steps to set up AI, MAX Viewer in your workspace:

1

Create the folder structure

In your workspace root, create the following folders:

mkdir -p Artifacts
2

Create an index.html file

Create a home page for your artifacts dashboard:

Artifacts/index.html
3

Reload VS Code window

Press Cmd+Shift+P and run "Developer: Reload Window" to restart the extension.

Example index.html

Here's a minimal template you can use for your artifacts home page:

<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>My Artifacts</title>
    <style>
        body {
            font-family: 'Inter', sans-serif;
            background: #000;
            color: #fff;
            padding: 40px;
        }
        h1 { color: #004BFF; }
        a { color: #35C2FF; }
    </style>
</head>
<body>
    <h1>My Artifacts</h1>
    <p>Your HTML artifacts will appear in the dropdown above.</p>
</body>
</html>

How It Works

  • Artifacts folder: Place your HTML files in Artifacts/
  • Auto-discovery: All HTML files are listed in the toolbar dropdown
  • Sorted by date: Newest files appear first
  • Home page: index.html is your dashboard landing page

AI Agent Integration

When AI agents (like Claude Code) create HTML artifacts, they can open them directly in the viewer:

open "vscode://aimax.aimax-viewer/openBrowser?http://127.0.0.1:3124/Artifacts/my-artifact.html"

This opens the artifact inside VS Code with the full toolbar (reload, navigation, dropdown selector).

Claude Bridge Skill

The aimax-bridge skill adds a floating panel to your HTML artifacts that lets you send prompts directly to Claude Code from the browser.

  • Copy — copies the prompt to clipboard
  • Copy & Terminal — copies and opens an interactive Claude session in VS Code terminal
  • Copy & VS Code — copies and opens the Claude Code extension panel
  • Ask Inline — sends the prompt to claude -p and shows the response directly in the artifact

Installation (Plugin)

From inside a Claude Code session, add the marketplace and install:

/plugin marketplace add https://github.com/maxturazzini/aimax-viewer.git
/plugin install aimax-bridge@aimax-viewer

Or for local development/testing:

claude --plugin-dir /path/to/aimax-viewer

This registers the aimax-bridge skill and makes /aimax-bridge available.

Installation (Manual)

  1. Copy the skills/aimax-bridge/ folder into your project's .claude/skills/ directory (or symlink it)
  2. In your HTML artifact, add the bridge snippet just before </body> (see the Bridge Demo for a working example)
  3. Serve the artifact through AI, MAX Viewer — the bridge uses the same HTTP server's /__claude endpoint

Once installed, invoke from Claude Code with /aimax-bridge when creating a new artifact.
